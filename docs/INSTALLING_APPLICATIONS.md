To be run in development mode applications must be signed with LG App Test:  https://developer.lge.com/apptest/retrieveApptestList.dev

* Login with your developer account and click New Test.

* Fill in the form. 

	App Title and App Icon could be anything convenient.

	URL must be either http url or local file path, relative to archive root. If the file does not exist the form will raise an error. 

* Download the packaged app and extract it onto NTFS formatted USB drive into `/lgapps/installed` directory.
